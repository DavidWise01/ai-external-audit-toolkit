# Forensic Convergence Protocol

1. Define the claim.
2. Label the layer: behavior / policy / tooling / public record / hidden internal.
3. Identify what can be observed directly.
4. Identify what requires external corroboration.
5. Run repeated prompts with fixed wording.
6. Log exact outputs and timestamps.
7. Compare for consistency and provenance drift.
8. Separate proved / inferred / speculative.
9. Re-run after time or context changes.
10. Update claim status.
